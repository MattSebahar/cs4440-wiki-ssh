Fundamental Operations

Here are SQL operations and below are examples of them in use using the SQL data tables at the bottom.

SELECT : return a result set of rows, from one or more tables

SELECT ID, Name from CourseStaff;

INSERT: insert data into table

INSERT INTO CourseStaff (Name, Role, City) VALUES (“Jane Doe”, “TA”, “Chicago”);

UPDATE: update data in table 

UPDATE CourseStaff SET Name = “Prof Nagy”, City = “New York City” WHERE id = 0;

DELETE: delete existing records 

DELETE FROM CourseStaff WHERE City="Salt Lake"

UNION: combine results of multiple queries

SELECT Name FROM CourseStaff UNION SELECT Name FROM Professors;

WHERE/AND/OR: conditional operations to find data in table

SELECT * FROM CourseStaff WHERE Name = “Ethan Quinlan”
SELECT * FROM CourseStaff WHERE City = “Salt Lake” AND Role = “TA”
SELECT * FROM Courses WHERE City = “Salt Lake” OR CourseID = "CS4440"

Syntactical Tips

Here are SQL syntax tips to keep in mind.

  • * : all
  • NULL : nothing
  • -- : comment-out the rest of the line (note the space at the end)
  • = : equal (comparison) operator
  • TRUE : true keyword
  • FALSE : false keyword
  • “1” = 1 : will result as FALSE
  • 1 = 1 : will result as TRUE
  • “1” = 1 OR 1 : will result in TRUE
